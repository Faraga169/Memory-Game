🎮 Memory Game (TypeScript)

A simple Memory Card Game built using HTML, CSS, and TypeScript with sound effects, scoring system, and shuffle logic.

🚀 Features
🎴 Flip card mechanism
🔄 Shuffle cards on restart
🎯 Score system (+10 for match, -2 for wrong attempt)
🔊 Sound effects (flip, match, wrong, win)
🏆 Win screen with final score
🔁 Play again functionality
🧠 Prevents double clicking using lock system
🛠️ Technologies Used
HTML5
CSS3
TypeScript (OOP)
DOM Manipulation
Audio API

🎮 How to Play
Click on any card to flip it
Try to find matching pairs
Each match gives +10 points
Wrong match gives -2 points
Match all cards to win the game
Click Play Again to restart
🧠 Game Logic
Cards are shuffled randomly at the start
Only 2 cards can be flipped at a time
If cards match → they stay open
If not → they flip back after 1 second
Game ends when all cards are flipped
🔊 Sounds
Flip sound when card is opened
Match sound when pair is correct
Wrong sound when pair is incorrect
Win sound when game is completed
